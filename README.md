# Steam Appeal Process Automator
## Description
The Steam Appeal Process Automator is a web application designed to automate the appeal process for Steam users who have been wrongfully banned.
## Prerequisites
* Node.js
* MongoDB
* Steam API key
## Installation
* Run `npm install` to install dependencies
## Environment setup
* Create a `.env` file with the required environment variables
## Run commands
* Run `npm run dev` for development mode
* Run `npm run build` and `npm run start` for production mode
