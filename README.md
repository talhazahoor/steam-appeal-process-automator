# Steam Appeal Process Automator

## Description
The Steam Appeal Process Automator is a web application designed to automate the appeal process for Steam users who have been wrongfully banned.

## Prerequisites
* Node.js
* MongoDB
* Redis

## Installation
1. Clone the repository
2. Run `npm install`

## Environment setup
1. Create a `.env` file and add your environment variables
2. Start the development server with `npm run dev`

## Run commands
* Development: `npm run dev`
* Production: `npm run build && npm run start`
